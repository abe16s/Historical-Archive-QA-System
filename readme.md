# Historical Archive QA System (Course Project)

Selected Topics: RAG-Based Projects — A retrieval-augmented QA system for historical documents with a FastAPI backend and Next.js frontend.

This project builds a RAG-based QA tool to answer questions from historical documents (archives, historical texts). It focuses on grounding answers in the provided sources and evaluating factual grounding to support trustworthy historical analysis.

## Project Structure
- `server/` – FastAPI backend (document upload/index, chat, evaluation)
- `frontend/` – Next.js frontend (upload, documents, chat, evaluation)

## How to Run (quick)
- Backend: see [server/README.md](server/README.md)
- Frontend: see [frontend/README.md](frontend/README.md)
